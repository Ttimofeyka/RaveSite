Hello, world!

You've probably seen this example on the main page of our website.
Well, let's see again! 


import <std/io>
void main {
    std::printIn("Hello world!");
}

Let's compile this code via the command line:

rave main.rave

This program outputs the string "Hello, world!" at console.

This example can be in any file with the extension ".rave".
This is a Rave's own extension that distinguishes your files from others.

First, we import the io file from the standard library.
It is necessary to work with the console and files.
If you are importing a file from the standard library, you need to use < and > near the conditional path.

Next is the declaration of the main function with the void type.
void indicates that the function should not return a value.
The main function must exist in programs that are not libraries and are bound to std. It is called automatically when the program starts.

std::println outputs our string to the console by adding a newline character to the end.

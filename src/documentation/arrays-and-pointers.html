Arrays and pointers

There is actually no big difference between pointers and arrays in Rave and another programming language - but it's still worth describing them here.

Arrays

Arrays are a linear collection of elements with a single type.
They help programmers to contain a large number of elements in a single variable.

Example of creating an array: 
        ushort[2] a;

Pointers

Pointers are variables that store the address where the object is located.
They are often used for any purpose.
Pointers are also a bit dangerous - an inexperienced programmer can easily work with them, which will lead to errors and bugs (undefined behavior).

Example of creating a pointer: 
        ushort[2] a;
ushort* b = &a;
In this example, using the prefix "&", we take the address of the variable "a" and set it as the value of the variable "b".

Portable

More features

• Speed

Since the beginning of its creation, Rave has been focusing on the performance of programs written on it. Thanks to the various flags (-rco, -O 3), you can easily reach the speed of the same program in C, or even surpass it.

• Safety

Rave is not a completely safety language, but there are a lot of language constructs (for example, SafePtr) that help programmers write safely code.

• No preprocessor

Rave does not have a built-in preprocessor. Instead, a compile time is used, which expands the capabilities of the language.

• Working with OOP

Rave has no classes, but it has structures that can be inherited, have methods, and so on.

Basic types

There are many different types in Rave, but only the most primitive ones are described on this page.


bool

Denotes a boolean data type.
It can have the value true or false.

void

Indicates that the function has no return value.

char

Denotes a character or number from -127 to 128.

uchar

Denotes a character or number from 0 to 255.

short

Denotes a number from -32,768 to 32,767.

ushort

Denotes a number from 0 to 65,535.

int

Denotes a number from −2,147,483,648 to 2,147,483,647.

uint

Denotes a number from 0 to 4,294,967,295.

long

Denotes a number from −9,223,372,036,854,775 808 to 9,223,372,036,854,775,807.

ulong

Denotes a number from 0 to 18,446,744,073,709,551,615.

cent

Denotes a number from −170,141,183,460,469,231,731,687,303,715,884,105,728 to 170,141,183,460,469,231,731,687,303,715,884,105,727.

ucent

Denotes a number from 0 to 340,282,366,920,938,463,463,374,607,431,768,211,456.

float

Denotes a number from 1.175494351 E - 38 to 3.402823466 E + 38.

double

Denotes a number from 2.2250738585072014 E - 308 to 1.7976931348623158 E + 308.
